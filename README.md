# Simple To-Do App

A full-stack web app to manage daily tasks built with Python, Flask, and SQLite.

## Features

- Add tasks
- Mark tasks as complete or undo them
- Delete tasks
- Tasks split into Pending and Completed sections

## Tech Stack

- Python
- Flask
- SQLite
- HTML + CSS

## How to Run

1. Clone the repository
   git clone https://github.com/yourusername/todo-app.git
   cd todo-app

2. Install Flask
   pip install flask

3. Run the app
   python app.py

4. Open your browser and go to
   http://127.0.0.1:5000
